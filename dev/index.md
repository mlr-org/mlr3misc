# mlr3misc

Package website: [release](https://mlr3misc.mlr-org.com/) \|
[dev](https://mlr3misc.mlr-org.com/dev/)

Miscellaneous helper functions for [mlr3](https://mlr3.mlr-org.com).

## Installation

Install the last release from CRAN:

``` r
install.packages("mlr3misc")
```

Install the development version from GitHub:

``` r
pak::pak("mlr-org/mlr3misc")
```
