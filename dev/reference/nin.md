# Negated in-operator

This operator is equivalent to `!(x %in% y)`.

## Usage

``` r
x %nin% y
```

## Arguments

- x:

  ([`vector()`](https://rdrr.io/r/base/vector.html))  
  Values that should not be in `y`.

- y:

  ([`vector()`](https://rdrr.io/r/base/vector.html))  
  Values to match against.
