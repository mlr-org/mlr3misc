# Capitalize the First Letter of Strings

Takes a character vector and changes the first letter of each element to
uppercase.

## Usage

``` r
capitalize(str)
```

## Arguments

- str:

  ([`character()`](https://rdrr.io/r/base/character.html)).

## Value

Character vector, same length as `str`.

## Examples

``` r
capitalize("foo bar")
#> [1] "Foo bar"
```
